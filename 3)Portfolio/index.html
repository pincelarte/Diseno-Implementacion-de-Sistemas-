Mi Portfolio

Soy Juan Cruz, estudiante de análisis de sistemas y desarrollador web. Me especializo en la creación de aplicaciones utilizando HTML, CSS y JavaScript, con un enfoque en diseño responsivo, accesibilidad y buenas prácticas. Actualmente continúo formándome en tecnologías modernas mientras busco aplicar mis conocimientos en proyectos desafiantes y seguir creciendo como profesional. Además, tengo interés en el trabajo en equipo, la resolución creativa de problemas y la mejora continua de mis habilidades técnicas. Me apasiona aprender nuevas herramientas y frameworks que me permitan desarrollar soluciones innovadoras y eficientes.

Proyectos

Backend

Desarrollé una API REST utilizando Node.js y Express para gestionar usuarios, autenticación y operaciones CRUD. El proyecto simula una plataforma de tareas con base de datos MongoDB, validación de entradas y manejo de errores. Ideal para prácticas de desarrollo backend moderno.

Front End

Diseñé una interfaz responsive utilizando únicamente HTML y CSS, enfocándome en la maquetación con Flexbox y Grid. El proyecto incluye animaciones suaves, un sistema de tarjetas con hover interactivo y una navegación adaptable para móviles, sin necesidad de JavaScript.

HTML

Como diseñador frontend, desarrollé una landing page completamente responsive utilizando HTML semántico y CSS personalizado. Me enfoqué en la jerarquía visual, tipografía, paleta de colores accesible y una estructura clara para mejorar la experiencia del usuario. El diseño equilibra estética y funcionalidad, con especial atención a los detalles y consistencia en todos los dispositivos.
